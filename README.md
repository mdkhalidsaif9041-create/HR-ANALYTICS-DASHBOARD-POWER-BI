# HR Analytics Dashboard (Power BI)

## Overview
This project presents an HR Analytics Dashboard built in Power BI to analyze employee attrition and workforce trends. The dashboard helps HR teams identify key factors affecting employee turnover and supports data-driven decision-making.

## Key Insights
- Employee Attrition Analysis
- Department-wise Attrition Trends
- Attrition by Age Group and Gender
- Job Role and Salary Analysis
- Overtime Impact on Attrition
- Employee Satisfaction Metrics

## Tools Used
- Power BI
- Microsoft Excel / CSV Dataset

## Dashboard Features
- Interactive Filters and Slicers
- KPI Cards
- Attrition Rate Visualization
- Employee Demographics Analysis
- Department and Job Role Breakdown

## Dataset
The dashboard is based on the `HR_Analytics.csv` dataset containing employee information such as demographics, job details, income, satisfaction levels, and attrition status.

## Objective
To provide actionable insights into employee retention and help organizations understand the factors contributing to employee attrition.
